macOS software

Find macOS software for every stage of brickfilming!

Production

You've got your script, set, and shots planned out in your head. Time to capture it!

  • Dragonframe

    The industry standard software for stop motion animation.

  • FrameByFrame

    Simple but free Stop Motion animation software for the Mac.

  • Framed

    Simple program that creates a video file from images that you import.

  • iStopMotion

    Frame capture software with many features. Prices vary.

  • SAM Animation

    An easy to use frame capture animation. Both free and paid version available.

  • StopMojo

    A free animation suite.

  • Boats Animator

    A stop motion animation program created using NW.js and WebRTC.

Post-production: Video

  • Blender

    CGI software that can be used for effects.

  • Cinefx

    Cinefx is an offline playback, editing, encoding, animation and visual effects tool based on the Jahshaka technology that allows you to create professional digital media on your desktop. Use the tools that hollywood uses to create blockbuster films and special effects.

  • Final Cut Pro X

    A professional video editor from Apple.
