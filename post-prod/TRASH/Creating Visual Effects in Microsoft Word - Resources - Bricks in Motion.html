Creating Visual Effects in Microsoft Word
Article by Dan O'Shea (Edit/Delete/Back to draft)

Here's a way to create visual effects using software that's already on your computer. It looks fairly cartoony, and takes a fair bit of patience.

EDIT: This was originally posted in the Mac section, because only Macs have Preview. There's probably a similar program for Windows/Linux, though. Preview is basically used here in the screen-capture parts.

Step 1

Assuming you have an animation ready, open up Microsoft Word and import the first photo that needs an effect. For this example, we will use a shot of a person shooting. We will edit in both a muzzle effect and a bullet casing flying out. Awesome.

OK. So you have your image in. Leave the folder you got it from open behind the Word document. Now, this frame needs the muzzle flash thing in it. There are lots of pre-keyed muzzle pictures on the web, I'm using the first one in this picture, though there are probably some better ones if you go looking. Basically, you put the muzzle flash over the image you imported in.

Step 2

Open up Preview (should come with your computer). Select File > Grab > Selection. This allows you to select part of your screen and turn it into a .tiff file. It needs to be the exact size of the photo. If there are any bits of white around it then it will turn out badly. It's usually pretty easy to get it right.

Step 3

OK, so we've done a muzzle shot, what now? Well, erase the picture and muzzle flash, and import the next photo in the animation sequence.

You'll have to create a bullet shell now, using autoshapes. Use the square, get the right shape and color (gradients are good), and then resize it. On Word, things can only go to a certain size and you can't get it smaller. One way to get around this is to group it with another object that is far away from it. You can make that group smaller, then ungroup it. I think that makes sense. Anyway, it's one way to get it to a good size. When it's the right size, put it over your frame.

Grab the frame on Preview.

Step 4

Delete that image, and import the next in the sequence. Make sure that the image is in the same position and is the same size (this should happen by default).

Move the bullet casing a little, grab the frame.

Repeat this until you're done.

Step 5

Put the frames from Preview into your animation. It should turn out something like this.

As you can see, the bullet casing looks alright but the muzzle flash was a bit of a fail.

Yes, that was a huge waste of time.

Anyway, experiment with it and you will do a much better job than me. Or just download some software. That's a way better idea.

And yes, this is a rather pointless article. I guess it serves as a reminder that you can do a lot of stuff with what you already have. Whatever. I bet you just read this to see what Microsoft Word sfx look like.
