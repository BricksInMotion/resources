How to add explosions in WAX
Article by Daniel K (Edit/Delete/Back to draft)

This article will show you how to add explosions to a video clip using WAX

Start by downloading WAX at http://www.debugmode.com/wax/, then download an exlposion from http://www.detonationfilms.com.  After downloading some stock footage, you will need to extract it using WinRAR, which can be downloaded here.  After that you will need to convert the video into an AVI format, because the video is encoded in an MOV format witch doesn’t work with WAX.  One of the best free video converters it called Format Factory, which can be downloaded here.  To convert it, open up Format Factory, and in the sidebar click all to AVI.  Then click add file and find the video.

After that click start at the top. It should start converting, and after it is done you can find the converted file in you documents folder inside the FFOutput folder.

After converting it, open up WAX.  Right click the little gray folder you see at the top and click Add Media Files.  Then add the converted stock footage (which can be found at the FFOutput folder in your documents).  You should also import the video that you want to put the explosion onto.  You should see both of the files in your project media.  Then drag both the videos into your timeline. Make sure the explosion video is in the first layer and the video you want your explosion added to, on the second layer.

Next, we're going to need to crop out the title from the explosion video that says Detonation Films.  Just slide the slider bar to were the explosion starts and just crop the video.

Now we need to chroma key the blue ans/or black out of the stock footage, and we can do that by clicking video plugins and adding the chroma key plugin to the explosion clip. Then we need to select the color picker tool and select the color we want to chroma key out.  Just click the color on the video preview, and you should see that most of the color was deleted but not all. To fix that, we need to slide the tolerance to delete more of the color.

Now if you play it you should see that the blue color is gone and there is only the explosion part.  Now we can adjust how big we want the explosion to be and were it is located.  To adjust the settings of the explosion video, we can add the 3d plugin to the explosion clip. In the side bar you should see a bunch of controls.  Using those controls, you should be able to adjust the settings of the explosion clip.
